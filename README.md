# 📖 Reading Visualization Website

一个基于 React 的个人年度阅读数据可视化网站，用于展示个人一年的阅读情况、阅读分析以及书籍总结。

## 📌 项目简介

本项目通过可视化的方式，对个人年度阅读数据进行整理和展示，包括阅读总览、年度推荐、阅读分析、书籍列表以及金句摘录等内容，适合作为个人作品展示或年度阅读总结页面。

### 技术栈

React+ECharts+tailwindcss+Styled Components

### 功能模块
#### 一、年度概览（Overview）

年度阅读书籍总数、阅读总时长、阅读天数、关键词 / 词云展示

用于快速了解一整年的阅读整体情况。

#### 二、年度 TOP3 推荐榜单

展示年度最推荐的三本书，以及书籍的详细信息

#### 三、阅读分析（Data Visualization）

通过不同图表从不同角度分析阅读情况

#### 五、书籍总览（All Books）

按春 / 夏 / 秋 / 冬四个阶段划分书籍

展示书籍的详细信息

#### 六、年度金句墙（Quote Wall）

展示年度阅读中记录的金句
