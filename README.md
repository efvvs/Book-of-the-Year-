
# 📖 Reading Visualization Website

一个基于 React 构建的个人年度阅读数据可视化网站。
通过图表与视觉设计，将一整年的阅读行为转化为可感知、可分析、可回顾的阅读报告页面。

> 把“读过很多书”这件事，变成可以被看见的数据。

---

## ✨ 项目亮点

* 📊 使用 ECharts 构建多维度阅读数据分析
* 🎨 TailwindCSS + Styled Components 实现现代化视觉风格
* 📈 数据可视化驱动内容表达，而不是静态罗列
* 🧩 模块化结构，便于扩展年度数据
* 🖱 交互细节设计（动画、悬浮效果）

---

## 🛠 技术栈

* React
* ECharts
* TailwindCSS
* Styled Components

---

## 📂 功能模块

### 1️⃣ 探索阅读之旅

以“地图交互”的形式呈现年度阅读结构。

页面中心为一张抽象化阅读地图，不同模块以标记点（Marker）的方式分布其上。
用户可点击不同标记点，快速跳转至对应功能模块。

### 2️⃣ 年度阅读概览

从宏观层面呈现年度阅读数据：

* 年度阅读总书籍数
* 阅读总时长
* 阅读天数
* 关键词 / 词云展示

用于快速建立“这一年我读了什么”的整体认知。

---

### 3️⃣ 年度 TOP3 推荐榜单

展示年度最推荐的三本书，包括：

* 书籍封面
* 作者
* 简要推荐理由
* 书籍具体的阅读信息

---

### 4️⃣ 阅读数据分析

通过图表从不同维度分析阅读行为，例如：

* 月度阅读趋势
* 阅读类型分布
* 阅读时间分布
* 阅读书籍状态
* 作者地区分布

使用 ECharts 实现数据动态可视化。

---

### 5️⃣ 书籍总览（All Books）

按 春 / 夏 / 秋 / 冬 四个阶段划分书籍。

展示：

* 书籍封面
* 作者
* 阅读时间
* 简短摘要

强化时间感与阅读轨迹。

---

### 6️⃣ 读后小记（Reading Notes）

用于展示每本书阅读后的即时文章卡片，可以跳转到微信读书。

展示：

* 卡片式布局
* 横向滚动

---

### 7️⃣ 年度金句墙

展示年度阅读中记录的金句。

* 横向滚动展示
* 鼠标悬浮动画展开全文
* 强调“阅读留下的痕迹”

---

## 🎯 项目意义

阅读是一种长期行为，但它往往缺少结构化回顾。

本项目尝试通过数据可视化的方式：

* 将阅读转化为可量化的数据
* 将阅读感受结构化表达
* 将个人阅读记录转变为可展示的作品

它既是阅读总结，也是一种数据表达实践。
